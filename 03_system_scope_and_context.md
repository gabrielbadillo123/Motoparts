# 03. System Scope and Context

## Contexto del ERP
El sistema interactúa con actores humanos y sistemas externos.

### Actores principales
- Vendedor, Encargado de Inventario, Administrador, Gerente.
- Cliente final (externo).

### Sistemas externos
- Servicio de facturación electrónica.
- Sistema contable.
